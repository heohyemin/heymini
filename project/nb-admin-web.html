NetworkBridge 관리자 웹 UI/UX 개선

운영 중인 보안 솔루션(NetworkBridge) 관리자 웹의 UI/UX를 개선한 프로젝트로, 실제 관리자 사용 흐름을 기준으로 화면 구조와 시각적 우선순위를 재정리했습니다.

#UI/UX디자인#Web#퍼블리싱

개선된 NetworkBridge 관리자 메인 화면

Project Summary

NetworkBridge 관리자 웹은 다양한 정책·상태·로그 정보를 다루는 보안 솔루션으로, 기능 중심으로 구성된 기존 화면에서 정보 인지와 작업 흐름에 어려움이 있었습니다. 본 프로젝트에서는 기획과 디자인에 참여해 관리자 사용 시나리오를 기준으로 UI 구조를 재정리하고, 일부 프론트 기능 개선 작업에도 함께 참여했습니다.

As-Is 문제점

  • 기능 중심 화면 구성으로 정보 밀집도가 높음
  • 상태·정책·로그 파악이 직관적이지 않음
  • 관리자 주요 액션의 시각적 위계가 불명확함
As-Is
To-Be

To-Be 개선 방향

  • 정보 성격별 그룹화 및 레이아웃 재정리
  • 상태·정책·사용 현황을 한눈에 파악할 수 있도록 UI 개선
  • 관리자 작업 흐름을 고려한 버튼 및 인터랙션 구조 설계

상태·정책 정보를 그룹화하여 관리자 판단 속도를 개선

결과

  • 관리자 판단 속도 및 작업 효율 향상
  • 운영 환경에 무리 없는 UI 개선
  • 실제 서비스에 반영되어 운영 중
