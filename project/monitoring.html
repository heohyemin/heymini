NB·FB·NDT 통합 모니터링

NB, FB, NDT 서비스를 통합 관리하기 위해 새롭게 설계한 통합 모니터링 대시보드 신규 프로젝트입니다. 기존 UI가 존재하지 않던 환경에서 관리 효율 향상을 목표로 정보 구조와 비주얼 시스템을 최초로 디자인했으며, 여러 서비스의 상태와 핵심 지표를 단일 인터페이스에서 통합적으로 탐색하고 모니터링할 수 있는 관리자 중심 UX/UI를 설계했습니다.

#UI/UX디자인 #Dashboard #신규프로젝트

통합 모니터링 대시보드 메인 화면

Project Summary

멀티 서비스의 상태 및 핵심 지표를 최초로 구조화한 통합 모니터링 대시보드 UI 설계 프로젝트입니다. 관리자 사용성, 지표 탐색 효율, 빠른 의사결정을 돕는 UX 흐름을 핵심 목표로 기획·디자인했습니다.

설계 핵심 포인트

  • 멀티 서비스 상태의 통합 시각화
  • 핵심 지표 중심 정보 구조 설계
  • 관리자 인지 부하 최소화 및 의사결정 가속 UX

결과

  • NB·FB·NDT 서비스 상태 및 지표를 단일 UI에서 통합 확인 가능
  • 최초 설계 기반의 구조적 대시보드 UI/UX 구축 경험 확보
