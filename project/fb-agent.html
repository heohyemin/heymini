개선된 FileBridge PC Agent 메인 화면

Project Summary

FileBridge PC Agent는 파일 전송 및 상태 관리를 수행하는 에이전트로, 기존 화면은 기능 위주 구성으로 사용자 흐름 파악이 어려웠습니다. 본 프로젝트에서는 실제 사용 시나리오를 기준으로 UI 구조를 재정리하고, 개발 적용을 고려한 디자인 가이드를 함께 제공했습니다.

  • 로그인·알림·설정 화면 간 시각적 일관성 부족
