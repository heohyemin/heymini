NDT 사용자 메인 화면 (PC 환경)

Project Summary

NDT 사용자 화면은 접속 상태 확인과 실행 작업이 빈번한 서비스로, 기존 UX에서는 상태 전달이 분산되어 사용 흐름 파악이 어려웠습니다. 본 프로젝트에서는 접속 → 확인 → 실행의 단계별 UX 흐름을 기준으로 화면 구조를 재정리했습니다.

As-Is 문제점

  • 상태 전달이 분산되어 현재 상황 인지가 어려움
  • 팝업 중심 UX로 사용 흐름이 끊김
  • 디바이스별 UI 밀도 차이 발생
