개선된 파일 전송 관리자 솔루션 로그인 화면

Project Summary

파일 전송 관리자 솔루션 로그인 화면은 첫 진입 지점이라 신뢰감과 집중도가 중요했습니다.
운영 중인 환경을 고려해 기존 스타일을 건드리지 않고, custom.css를 추가하는 방식으로 UI 개선과 퍼블리싱을 함께 진행했습니다.

As-Is 문제점

  • 시각적 위계가 약해 입력 영역과 주요 액션이 한눈에 들어오지 않음
  • 보안 솔루션 특성에 필요한 ‘안정감/신뢰감’이 충분히 전달되지 않음
  • 운영 중 서비스로 인해 기존 CSS 변경 시 리스크가 큼

To-Be 개선 방향

  • 입력/버튼 영역의 우선순위를 재정리해 가독성과 집중도 강화
  • 보안 제품 톤에 맞는 안정적인 레이아웃과 시각 언어로 리디자인
  • 기존 CSS는 유지하고 custom.css로만 덮어써서 롤백 가능한 적용 방식 선택

결과

  • 첫 진입 화면의 신뢰감과 정보 전달력 강화
  • 입력/액션 구조가 정리되어 사용 흐름이 명확해짐
  • custom.css 제거 시 원복 가능한 구조로 운영 안정성 확보
