무료체험 페이지 퍼블리싱을 100% 담당한 프로젝트로, 기존 디자인을 기반으로 반응형 웹 퍼블리싱을 진행했습니다.

Web 반응형 퍼블리싱

무료체험 페이지 메인 화면

Project Summary

퍼블리셔 부재로 인해 유지보수 공백이 발생한 상황에서, 기존 디자인을 기반으로 반응형 퍼블리싱을 전면 적용했습니다. 다양한 디바이스 환경에서 동일한 경험을 제공하는 것을 목표로 UI 구현을 안정적으로 정리했습니다.

모바일/태블릿 환경까지 고려한 반응형 UI 적용

섹션별 UI 규칙을 체계화하여 구현 안정성을 높이고, AOS 라이브러리를 기반으로 인터랙션을 적용했습니다.

결과

  • 반응형 퍼블리싱 전면 적용
  • 다양한 디바이스 환경 대응
  • 디자인 의도에 맞는 UI 구현 정리
