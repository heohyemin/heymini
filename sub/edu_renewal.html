교육 서비스 메인 리뉴얼

운영 중인 가상머신(VM) 사용자 화면 UI/UX 개선 프로젝트로, PC·태블릿 환경에서 실제 사용 시나리오를 기준으로 화면을 재구성했습니다.

UI/UX 디자인 반응형

가상머신 사용자 메인 화면 (PC 환경)

Project Summary

교육 서비스 메인 페이지는 콘텐츠가 많고 업데이트가 잦아, 사용자가 핵심 정보를 빠르게 스캔하고 다음 행동으로 이동할 수 있는 구조가 중요했습니다. 본 프로젝트에서는 콘텐츠 우선순위와 시각적 위계를 재정리하고, 서비스 톤을 유지하면서도 더 명확한 정보 구조로 리뉴얼했습니다.

As-Is 문제점

  • 콘텐츠가 분산되어 핵심 정보 스캔이 어려움
  • 프로모션/공지/핵심 영역의 위계가 불명확함
  • 반응형 환경에서 UI 밀도 조정이 필요함

To-Be 개선 방향

  • 콘텐츠 성격별 섹션 구조 재정리
  • 핵심 CTA 중심의 시각적 우선순위 강화
  • 반응형 환경에서 정보 밀도 균형 조정
